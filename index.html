UtilsKit

轻量前端工具函数库，零依赖，浏览器&Node双端通用，内置格式化、数组处理、存储工具。

技术栈

TypeScript ESModule Tree-Shaking

快速使用

npm install utils-kit
import { formatTime } from 'utils-kit'
console.log(formatTime(Date.now()))

LiteUI

极简轻量UI组件库，无第三方样式依赖，原生CSS，适配移动端与管理后台。

技术栈

Vanilla JS CSS Variables 响应式

快速使用

import { Button, Modal } from 'lite-ui'
Modal.show({title:'提示',content:'演示弹窗'})

FetchMini

基于原生fetch封装的请求库，拦截器、缓存、超时、自动序列化，替代axios轻量化方案。

技术栈

Fetch API 缓存策略 请求拦截

快速使用

import http from 'fetch-mini'
const res = await http.get('/api/list')
文档 Github仓库

FileHelper

浏览器端文件处理工具，图片压缩、文件导出、Blob转换、分片上传封装。

技术栈

Blob FileReader 图片压缩

快速使用

import { compressImg } from 'file-helper'
compressImg(file, {quality:0.7})
